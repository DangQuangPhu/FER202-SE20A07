FER202 PRACTICAL EXAM
Student Feedback & Course Review System

1. Project Setup
This project was initialized using Vite with React.
State management: Redux Toolkit.
Routing: React Router DOM.
HTTP Client: Axios.
Mock Backend: JSON Server.
Styling: Bootstrap and Custom CSS.

2. Installed Packages
- react, react-dom
- react-router-dom
- @reduxjs/toolkit, react-redux
- axios
- bootstrap, bootstrap-icons
- json-server (dev dependency for backend)

3. How to Run the Application
Step 1: Install dependencies
Run the following command in the terminal:
npm install

Step 2: Start the JSON Server (Backend)
Run the following command to start the mock database on port 3001:
npm run server

Step 3: Start the React Development Server (Frontend)
Open a new terminal window/tab and run:
npm run dev

Step 4: Access the Application
The frontend will start on port 5173 (or 5174/5175 if 5173 is occupied).
Open your browser and navigate to the local URL provided by Vite (e.g., http://localhost:5173).
